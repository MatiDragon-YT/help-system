**El servicio de lenguajes** es una de las características de Sanny Builder que brinda una rica experiencia de desarrollador. Es responsable de la extracción continua de datos del documento abierto y todos los archivos importados. Es importante saber que la minería funciona en un subproceso separado y no refleja de inmediato lo que acaba de escribir. Podría haber un retraso notable mientras el servicio de lenguajes reacciona ante un cambio en el documento. Si por algún motivo te distrae o no estás satisfecho con el rendimiento del servicio, puedes desactivarlo en las [opciones](options/editor.md#editor-configuration). Para una mejor capacidad de respuesta, el servicio de lenguajes solo escanea una porción del documento actual antes de la línea activa, no el archivo completo. Esto está controlado por el valor [Rango de escaneo](options/editor.md#code-scan-distance) establecido en las opciones del Editor. Cuanto mayor sea el valor, más líneas escanea el servicio y más símbolos encuentra, pero también aumenta el tiempo necesario para que el editor refleje los cambios realizados en el documento. Si tiene una estación de trabajo menos poderosa, considere usar el valor predeterminado de `250` líneas. Desde v3.7.0, el servicio de lenguaje es capaz de encontrar [constantes definidas](coding/constants.md#syntax) y sus valores. Esta información luego está disponible para el editor para [resaltar](options/editor.md#color-theme) y para [mostrar valores](features.md#displaying-information-about-opcode) de constantes. ```sb3 const x = 10 end x // x queda resaltada ``` El resaltador de sintaxis ofrece dos formas de colorear las constantes: * resaltar todas las constantes usando el mismo conjunto de reglas * aplicar las reglas que de otro modo se aplicarían al valor constante ("resaltado semántico") Con el resaltado semántico habilitado y dado el ejemplo anterior, `x` se colorearía como un número normal (el color granate por defecto). El servicio de idioma también sirve como proveedor de datos para la función de autocompletar cuando presiona `Ctrl+Espacio`. Muestra una lista de constantes y sus valores. Cuando el servicio de idioma está deshabilitado, no se muestran constantes en la lista. {% hint style="info" %} Lee más detalles técnicos sobre la implementación del servicio aquí: * [Notes on constants highlighting.
