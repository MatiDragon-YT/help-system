Puede llamar a esta herramienta presionando `Ctrl+Alt+2` o seleccionando el elemento del menú: `Herramientas->Herramientas del IDE->Buscador de Opcodes`. :::c ![](../img/ost.png) ::: Para encontrar un opcode, escriba algunas palabras en la línea de entrada, p. `actor car`. La herramienta muestra los opcodes con estas palabras. También puede utilizar operadores de búsqueda especiales. Cuando la herramienta está abierta, comprueba si hay una palabra seleccionada en el editor. Si se encuentra la palabra seleccionada, se copiará en el campo de búsqueda. ## Copiar Resultados Para copiar un opcode en el portapapeles, selecciónelo en la lista y presione `Enter`.\ Para agregar otro opcode al contenido del portapapeles, presione `Shift+Enter`.\ Para copiar toda la lista de resultados, presione `F2`. ## Atajos Combinados `Enter` copia el opcode seleccionado en el portapapeles\ `Shift+Enter` añade el opcode seleccionado al portapapeles\ `F1` - mostrar información de ayuda\ `F2` - copia todos los opcodes de la lista de resultados en el portapapeles\ `F3` - ordenar la lista por opcodes\ `F4` - ordena la lista alfabéticamente\ `F11` - borra el campo de búsqueda, muestra todos los opcodes\ `ESC` - cerrar la ventana de herramientas ## Consultas de Búsqueda ### Operador AND Un solo espacio entre palabras sirve como el operador `AND`: ``` @ player ``` Encuentra todos los opcodes con `@` y la palabra `player`. ### Operador OR El carácter vertical `|` es el operador `OR`: ``` @ | player ``` Encuentra todos los opcodes con `@` o `player`. Si escribe `|` como primer carácter en la búsqueda, la herramienta conectará todas las siguientes palabras con el operador OR. ``` | actor player car ``` Encuentra opcodes con `actor`, `player` o `car`. ### Operador NOT Dos guiones antes de la palabra excluyen los opcodes con esta palabra del resultado: ``` car --actor ``` Encuentra opcodes con `car` pero sin `actor`. Se puede usar un solo guión antes de los identificadores o los caracteres `$` y `@`, pero no los números: ``` -10 -@ -car ---1 ``` Encuentra opcodes con el número `-10` y sin `@`, `car` y `-1` ### Buscar en opcodes condicionales `^` - muestra solo opcodes condicionales\ `^word` - encuentra `palabra` en los opcodes condicionales: ``` ^car == ``` Encuentra todos los opcodes condicionales con `car` y `==`. `-^` - excluye todos los opcodes condicionales de la lista.: ``` player -^ ``` Encuentra todos los opcodes no condicionales con `player`. `-^word` - excluye los opcodes condicionales con `word`: ``` player -^actor ``` Encuentra opcodes con `player`, con la excepción de los opcodes condicionales con `actor`. ### Respeta el orden de las palabras. `%` - encuentra opcodes con las palabras en el orden dado: ``` % @ = @ ``` Encuentra opcodes donde `@` `=` `@` se suceden (posiblemente con otras palabras en medio).
