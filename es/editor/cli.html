## --compile `--compile <ruta de entrada> [ruta de salida]` - compila un archivo y se cierra ``` sanny.exe --compile C:\myscm.txt C:\myscm.scm ``` {% hint style="info" %} El parámetro `ruta de salida` es opcional. Si no hay un parámetro de ruta de salida, el compilador usa las [reglas de formato](options/formats.md#file-name-format) para calcular el nombre del archivo de salida. {% endhint %} {% hint style="info" %} El archivo de entrada no debe tener errores. Si se encuentran errores durante la compilación, se registrarán en el archivo `compile.log`. Cuando finaliza la compilación, el programa se cierra. {% endhint %} Atajo: `-c` ## --debug `--debug` - alterna [opciones de depuración](console.md#running-with-debug) ``` sanny.exe --debug 110000 ``` ## --game `--game <game>` - selecciona el [modo predeterminado](../../edit-modes/README.md#type) para el juego dado. Los valores conocidos de `game` son: * `gta3` * `vc` * `sa` * `lcs` * `vcs` * `vc_mobile` * `sa_mobile` Atajo: `-g` ``` sanny.exe --game sa ``` ## --mode `--mode <id>` - selecciona el modo por `id`. Los [modos](../../edit-modes/README.md) y sus ID están definidos en el archivo `modes.xml`. Atajo: `-m` ``` sanny.exe --mode vcs_psp ``` ## --no-splash `--no-splash` - ejecuta Sanny Builder sin la pantalla de inicio ``` sanny.exe --no-splash ``` ## -x `-x <file path>` - Establece la ruta al archivo con la configuración del [modo de edición](../../edit-modes/README.md) ``` sanny.exe -x ..\my-modes.xml ``` ## --option {% hint style="info" %} Disponible desde v3.8.0 {% endhint %} `-o <nombre de la opción> <valor>` - establece el valor de la [opción](options/README.md) especificada. Para establecer varias opciones, enumere cada una de ellas individualmente: ``` -o <nombre de la opción1> <valor de la opción1> -o <nombre de la opción2> <valor de la opción2> ... -o <nombre de la opciónN> <valor de la opciónN>``` El nombre de la opción puede ser uno de los siguientes: {% hint style="warning" %} Esta lista no es definitiva y está sujeta a cambios en cualquier momento. {% endhint %}
