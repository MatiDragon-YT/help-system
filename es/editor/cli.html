## --compile `--compile <ruta de entrada> [ruta de salida]` - compila un archivo y se cierra ``` sanny.exe --compile C:\myscm.txt C:\myscm.scm ``` {% hint style="info" %} El parámetro `ruta de salida` es opcional. Si no hay un parámetro de ruta de salida, el compilador usa las [reglas de formato](options/formats.md#file-name-format) para calcular el nombre del archivo de salida. {% endhint %} {% hint style="info" %} El archivo de entrada no debe tener errores. Si se encuentran errores durante la compilación, se registrarán en el archivo `compile.log`. Cuando finaliza la compilación, el programa se cierra. {% endhint %} Atajo: `-c` ## --decompile `--decompile [output file path]` - descompila el archivo y se cierra. ``` sanny.exe --decompile C:\myscm.scm C:\myscm.txt ``` {% hint style="info" %} El parámetro `output file path` es opcional. Si no hay ningún parámetro de ruta de salida, el compilador utiliza las [reglas de formato](options/formats.md#file-name-format) para calcular el nombre del archivo de salida. {% endhint %} {% hint style="info" %} El archivo de entrada no debe contener errores. Si se encuentran errores durante la descompilación, se registrarán en el archivo `compile.log`. Cuando finaliza la descompilación, el programa se cierra. {% endhint %} Shortcut: `-d` **--use-source** - argumento opcional que hace que el descompilador utilice el código fuente [incrustado en el script](options/general.md#add-extra-info-to-scm) (si está presente). Por defecto, el código fuente se ignora. ``` sanny.exe --decompile C:\1.cs --use-source ``` ## --debug `--debug` - alterna [opciones de depuración](console.md#running-with-debug) ``` sanny.exe --debug 110000 ``` ## --game `--game <game>` - selecciona el [modo predeterminado](../../edit-modes/README.md#type) para el juego dado. Los valores conocidos de `game` son: * `gta3` * `vc` * `sa` * `lcs` * `vcs` * `vc_mobile` * `sa_mobile` Atajo: `-g` ``` sanny.exe --game sa ``` ## --mode `--mode <id>` - selecciona el modo por `id`. Los [modos](../../edit-modes/README.md) y sus ID están definidos en el archivo `modes.xml`. Atajo: `-m` ``` sanny.exe --mode vcs_psp ``` ## --no-splash `--no-splash` - ejecuta Sanny Builder sin la pantalla de inicio ``` sanny.exe --no-splash ``` ## -x `-x <file path>` - Establece la ruta al archivo con la configuración del [modo de edición](../../edit-modes/README.md) ``` sanny.exe -x ..\my-modes.xml ``` ## --option {% hint style="info" %} Disponible desde v3.8.0 {% endhint %} `-o <nombre de la opción> <valor>` - establece el valor de la [opción](options/README.md) especificada. opciónN> <valor de la opciónN>``` El nombre de la opción puede ser uno de los siguientes: {% hint style="warning" %} Esta lista no es definitiva y está sujeta a cambios en cualquier momento. {%
