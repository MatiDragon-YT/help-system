Aquí puede cambiar la configuración del IDE. :::l ![](../../img/options-editor-en.png) ::: ## Tema de Color :::l ![](../../img/sb-options-color-theme.png) ::: Este selector permite elegir uno de los temas predefinidos para personalizar los colores de la interfaz y resaltar la sintaxis. El botón Editar inicia [Editor de Temas](theme-editor.md). ## Distancia de Escaneo de código :::l ![](../../img/editor-code-scan-en.png) ::: El rango de escaneo de código se usa cuando presiona `Ctrl+Espacio` para encontrar [constantes](../../coding/constants.md), etiquetas y [variables](../../coding/variables.md ). El valor le dice al editor cuántas líneas escanear hacia arriba y hacia abajo a partir de la línea activa. Si establece un número de `0`, el editor escanea todo el archivo fuente. Si la opción `Lista automática` está marcada, la lista de etiquetas, modelos y variables disponibles aparece automáticamente a medida que escribe `@`, `$`, `#`. ## Fuente :::l ![](../../img/editor-font-en.png) ::: Aquí puede elegir la fuente y configurar su visualización. Sanny Builder solo admite [fuentes monoespaciadas](https://en.wikipedia.org/wiki/Monospaced_font). `AaBbCcDd` refleja la configuración seleccionada. ## Configuración del Editor :::l ![](../../img/sb-editor-configuration.png) ::: `Mostrar panel principal` – cambiar la barra de herramientas superior\ `Mostrar información de opcodes` – hacer que el editor muestre [diferente información](../features.md#displaying-information-about-opcode) en la barra inferior\ `Números de linea` – muestra los números de línea en el área de relleno izquierda\ `Mostrar borde del canalón` – mostrar la línea de borde en el lado derecho del área de relleno izquierda. Desmarca esta opción para aumentar visualmente el espacio disponible\ `Cargar el último archivo cerrado al iniciar` – abrir el último archivo cerrado al inicio\ `Cargar todos los archivos cerrados` – abre todos los archivos cerrados al inicio. Esta opción solo está disponible cuando la opción Cargar el último archivo cerrado al iniciar\ `Confirmar cuando exista` – confirmar la salida para evitar la terminación ocasional del programa\ `Habilitar servicio del lenguaje` - cambie el [servicio de idioma](../language-service.md). Deshabilitar este servicio también hace que el resaltado de contenido semántico no esté disponible. No afecta a un documento que ya está abierto en el editor\ `Resaltador semantico` - se aplican las mismas reglas a las constantes que a sus valores. Se ignorarán las reglas del tema de color para el resaltado constante..
