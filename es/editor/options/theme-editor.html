Editor de Tema es una nueva herramienta integrada que reemplaza las opciones de resaltado de sintaxis. Se puede iniciar desde la pestaña [Editor](editor.md#color-theme). :::l ![](../../img/theme-editor-en.png) ::: Theme configuration includes colors and style for interface and code elements. Cada elemento tiene 3 propiedades: `Carácter`, `Fondo` y `Estilo`. Algunas propiedades pueden no estar disponibles para ciertos elementos.\ Los elementos de estilo disponibles son: - **B** – bold (negrita) - **I** – italic (cursiva) - **U** – underlined (trasado) ### **Elementos de la Interfaz** * **Editor** – color del texto y fondo del área principal de edición * **Línea Activa** – una línea donde está el cursor * **Canalón** – área a la izquierda del edito * **Panel Inferior** – barra de estado en la parte inferior del editor * **Bordes del Panel Inferior** – color de los bordes de sección en el panel inferior ### Elementos del Código * [Comentarios](../../coding/comments.md) – un texto después de `//` o entre `{}` o `/* */` * [Etiquetas](../../coding/data-types.md) – identificadores que comienzan con `@` * [Variables](../../coding/variables.md) – identificadores que comienzan con `$` (variables globales) o terminan con `@` (variables locales)\ También el tipo [aDMA](../../coding/data-types.md) (comienza con `&`) y [arrays](../../coding/arrays.md) se resaltan como variables. * [Palabras Claves](../../coding/keywords.md) – las palabras especiales almacenadas en el archivo `keywords.ini` (uno para todos los [modos de edición](../../edit-modes/README.html)) y un archivo `keywords.txt` (uno por modo de edición) * Números – números enteros y de punto flotante * Cadenas – un texto entre comillas dobles `" "` o comillas simples `' '` * Modelos: identificadores que comienzan con `#` * [Clases](../../coding/classes.md) – la parte de un comando de clase antes del carácter `.` * [Comandos](../../coding/classes.md) – la parte de un comando de clase después del carácter `.`, o una [subrutina](../../coding/data-types.md#etiquetas) * [Directivas ](../../coding/directives.md)– un comando que comienza con `{$` * [Constantes](../../coding/constants.md) – identificadores, declarados dentro de la construcción `CONST..END`. Ver también [Resaltado semántico](editor.md#editor-configuration). :::info La configuración de resaltado también se usa cuando el código fuente se guarda en los formatos `RTF` o `HTML`. :::
