Puede insertar un fragmento de código predefinido llamado plantilla de código escribiendo el nombre de la plantilla y presionando `F2`. Para mostrar la lista de plantillas disponibles, presione `Ctrl+J`. Por ejemplo, si escribe `for` y presiona `F2`, el editor agrega: ```sb3 for = to // step 1 end // for ``` ## Tipos de Plantillas Hay dos tipos de plantillas: *compartidas* y *exclusivas*. Las plantillas compartidas están disponibles en todos los [modos de edición](README.md). Estos están contenidos en el archivo `data\templates.txt`. Las plantillas exclusivas están diseñadas específicamente para el modo de edición actual. Se puede personalizar una ruta al archivo de plantilla exclusivo con la tecla [`template`](README.md/#templates) en `modes.xml`: ``` <templates type="default">@sb:\data\gta3\templates.txt</templates> ``` El editor fusiona el contenido de las plantillas exclusivas y compartidas en una sola lista. Las plantillas exclusivas tienen prioridad sobre las compartidas con el mismo nombre. ## Sintaxis del Archivo Cada archivo de plantilla tiene la siguiente sintaxis:\ El nombre de una plantilla se escribe en una línea separada. El signo igual `=` sigue al nombre. Después del signo igual, también puede agregar una breve pista. El código de la plantilla comienza en la siguiente línea, cada línea comienza con el signo igual. La posición del cursor está marcada con el carácter `|`. ## Añadir una Nueva Plantilla También es posible agregar una nueva plantilla desde el editor. Seleccione el código y haga clic secundario en el menú `Añadir nueva plantilla`. :::center ![](../img/ide-add-template-en.png) ::: Ingrese un nombre de la plantilla, una descripción (opcional) en la ventana de diálogo, elija el tipo de plantilla y presione el botón `OK`. La plantilla está lista para usar. Las plantillas personalizadas se guardan en `data\templates.custom.txt` (*plantillas personalizadas compartidas*) o en `data\<modo de edición>\templates.custom.txt`(*plantillas exclusivas personalizadas*). Este último se puede personalizar en la [configuración de modos](README.md#templates). {% hint style="info" %} La opción de guardar una plantilla solo para el modo actual está disponible si `modes.xml` tiene una ruta de archivo definida en la etiqueta `<template type="custom">`. {% endhint %} {% hint style="info" %} Una plantilla con el nombre `CLEO` desbloquea el elemento "Nuevo CLEO Script..." en el menú principal. El editor utiliza esta plantilla como contenido para un archivo recién creado. {% endhint %}
