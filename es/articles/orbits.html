Estas funciones permiten definir una trayectoria orbital para un objeto, donde se puede cambiar la distancia del radio a medida que se mueve por una trayectoria. Así, se puede simular diferentes tipos de órbitas, como elipses o círculos, y cambiar la distancia del objeto al punto central en cualquier momento de la órbita. # 2D ```math x = cos(a) \* r + cx y = sin(a) \* r - cy ``` Donde: * `r` es el radio inicial de la órbita alrededor del punto. * `a` es el ángulo de rotación del objeto. * `cx`, `cy` son las coordenadas del punto alrededor del cual orbita el objeto. ```sb3 // traducido a sb3 float 0@ {a} = 90.0 float 1@ {r} = -7.5 2@ {x} = Math.Cos(0@ {a}) 2@ {x} *= 1@ {r} 2@ {x} += 3@ {cx} 4@ {y} = Math.Cos(0@ {a}) 4@ {y} *= 1@ {r} 4@ {y} -= 5@ {cy} ``` # 3D ```math x = sin(ax) \* cos(ay) \* r + cx y = sin(ax) \* sin(ay) \* r + cy z = cos(ax) \* r + cz ``` Donde: * `r` es el radio inicial de la órbita alrededor del punto. * `ax` es el ángulo de rotación en el eje x. * `ay` es el ángulo de rotación en el eje y. * `cx`, `cy` y `cz` son las coordenadas del punto alrededor del cual orbita el objeto. ```sb3 // traducido a sb3 float 0@ {ax} = 90.0 float 1@ {ay} = 100.0 float 2@ {r} = -7.5 5@ {x} = Math.Sin(1@ {ay}) 6@ {x2} = Math.Cos(0@ {ax}) 5@ {x} *= 6@ {x2} 5@ {x} *= 2@ {r} 5@ {x} += 7@ {cx} 8@ {y} = Math.Sin(1@ {ay}) 9@ {y2} = Math.Sin(0@ {ax}) 8@ {y} *= 9@ {y2} 8@ {y} *= 2@ {r} 8@ {y} += 10@ {cy} 3@ {z} = Math.Cos(1@ {ay}) 3@ {z} *= 2@ {r} 3@ {z} += 4@ {cz} ``` --- :::gray Original post: _By MatiDragon_ :::
