Una **palabra clave** es una sola palabra que reemplaza el opcode para comandos simples. Por ejemplo, la palabra clave `WAIT` se puede utilizar en lugar del opcode `0001`. Para demostrar: ```sb3 wait 0 ``` en vez de ```sb3 0001: wait 0 ``` La lista de palabras clave se puede mostrar presionando `Ctrl+Space`.
