El operador `IF` evalúa una o más condiciones y crea una nueva rama de código ## Declaraciones IF de Bajo Nivel `00d6: if `\ ` `\ ` `\ `...`\ ` `\ `004D: jump_if_false `\ \ `N` significa el número total de condiciones dentro de la declaración IF y la forma en que ocurre la evaluación de las condiciones. |N|Número de Condiciones|Operador Lógico| |-| |0|1|la declaración IF es verdadera (`true`) sí la condición es verdadera| |1..7|2..8|`AND` (todas las condiciones deben ser verdaderas para que la declaración IF sea verdadera)| |21..27|2..8|`OR` (al menos una de las condiciones debe ser verdadera para que la declaración IF sea verdadera)| {% hint style="info" %} Una declaración IF puede contener hasta 8 condiciones. {% endhint %} {% hint style="info" %} Sanny Builder permite omitir el `0` despues del IF.\ `IF 0` y `IF` son equivalentes. {% endhint %} - ``- el nombre de la [etiqueta](data-types.md#labels) donde el script saltara si la declaración IF es falsa (`false`). - `` - cualquier opcode condicional que evalúe como `true` o `false` Si usted tiene las `Chequeo de condiciones` habilitado en las [opciones](../editor/options/general.md#check-conditions), se puede sustituir el número si con las palabras clave `AND` y `OR`. El compilador calcula el valor correcto por sí mismo. ```sb3 if and $var > 0 $var2 == 10.0 jf @anywhere ``` El compilador escribe el número `1` en lugar de `and`.\ \ `IF AND` - condiciones conectadas con el operador lógico `AND` (en reemplazo para `if 1..7`)\ `IF OR` - condiciones conectadas con el operador lógico `OR ` (en reemplazo para `if 21..27`) ## Constructores de Alto Nivel Para facilitar las condiciones de escritura, existen construcciones de alto nivel que no requieren etiquetas adicionales:\ `IF /AND/OR`\ ` `\ ` `\ ` ...`\ ` `\ `THEN`\ ` `\ `END`

`IF /AND/OR`\ ` `\ ` `\ ` ...`\ ` `\ `THEN`\ ` `\ `ELSE`\ ` `\ `END`\ \ Una condición es creada con las reglas descritas de las [condiciones de bajo nivel](conditions.md#low-level-if-statements). Después de `THEN` debe especificar los comandos que se ejecutarán si se cumple la condición. Después de `ELSE` debe especificar los comandos que se ejecutan si no se cumple la condición. La declaración `IF` se cierra con la palabra` END`. ```sb3 if $var == 5 then Inc($var) else Dec($var) end ``` :::info La opción de [`Chequeo de condiciones`](../editor/options/general.md#check-conditions) debe estar habilitada. ::: :::info Se admiten declaraciones IF anidadas. ::: ## Operadores relacionales * A **`==`** B - A es igual a B * A **`>=`** B - A es mayor o igual a B * A **`>`** B - A es mayor que B * A **`<`** B - A es menor que B * A **`<=`** B - A es menor o igual a B * A **`<>`** B - A es desigual a B `A` y` B` son operandos. El compilador puede averiguar un opcode si uno de los operandos es un número o una [cadena literaria](data-types.md#string-literals), o ambos [tipo de variables](variables.md#var-end-construct) son conocidas.
