Los siguientes comandos están integrados directamente en el compilador y sirven principalmente como [azúcar sintáctico](https://en.wikipedia.org/wiki/Syntactic_sugar). ## INC Incrementa el primer parametro por el segundo. Elprimer parametro es una [variable](variables.md). ```sb3 Inc($IntVariable, $Value) = $IntVariable += $Value ``` El segundo parametro es igual a `1` si no se especifica. ```sb3 Inc(1@) = 1@ += 1 ``` Desde la versión 3.06 es posible utilizar el operador `++` que se puede aplicar a las variables e incrementa su valor en uno. ```sb3 $var++ = $var += 1 ``` ## DEC Disminuye el primer parametro por el segundo. El primer parametro es una variable. ```sb3 Dec($IntVariable, $Value) = $IntVariable -= $Value ``` El segundo parametro es igual a `1` si no se especifica. ```sb3 Dec(1@) = 1@ -= 1 ``` Desde la versión 3.06 es posible utilizar el operador `--` que se puede aplicar a las variables y reduce su valor en uno. ```sb3 $var-- = $var -= 1 ``` ## MUL Multiplica el primer parámetro por el segundo. El primer parámetro es una variable. ```sb3 Mul($IntVariable, $Value) = $IntVariable = $IntVariable * $Value ``` El segundo parámetro es igual a `2` si no se especifica. ```sb3 Mul(1@) = 1@ = 1@ * 2 ``` ## DIV Divide el primer parámetro por el segundo. El primer parámetro es una variable. ```sb3 Div($IntVariable, $Value) = $IntVariable = $IntVariable / $Value ``` El segundo parámetro es igual a `2` si no se especifica. ```sb3 Div(1@) = 1@ = 1@ / 2 ``` ## ALLOC Esta función establece el desplazamiento de una variable global en el espacio de la [variable global](variables.md#variables-globales) que existe al principio del encabezado `main.scm` Esto solo es significativo para las variables personalizadas (p.ej. `$text`) no definidas en `CustomVariables.ini`. Las variables DMA siempre obtienen su desplazamiento en función del número en sus nombres, p.ej la variable `$40` siempre ocupa cuatro bytes en el desplazamiento `160` (40*4). El primer parámetro debe ser una variable global, el segundo parámetro debe ser un número entero positivo o cero. ```sb3 Alloc($MyVar, 40) - la variable $MyVar pasa a compilarse como $40 ``` {% hint style="info" %} Vea también `help\examples\alloc.txt` {% endhint %} ## SQR Multiplica la variable por sí misma. ```sb3 sqr($var) = $var *= $var ``` {% hint style="info" %} El tipo de variable debe ser [declarado](variables.md#constructor-var..end). {% endhint %} ## RANDOM Esta funcion genera un numero aleatorio dentro del rango especificado. ```sb3 $rnd = random(1, $high) ``` Esta función se puede utilizar para variables enteras y decimales. El opcode es seleccionado según el tipo de la variable resultada (`$rnd` en este ejemplo).
