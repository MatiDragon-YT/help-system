Una **constante** es un identificador con un valor predefinido. A diferencia de una variable, el valor de la constante no se puede cambiar en tiempo de ejecución. En tiempo de compilación, la constante se reemplaza por el valor asociado a ella. El valor de la constante puede ser numérico (números, [nombres de modelo](data-types.md#model-names) o [etiquetas](data-types.md#labels)) y una cadena (literales de cadena), y contener un expresión. Las constantes se declaran estática o dinámicamente. Cada [modo de edición](../edit-modes/README.md) uede cargar definiciones de constantes estáticas desde un archivo usando el parámetro `` en el `modes.xml`. Las declaraciones dinámicas se crean en el código del script con la sintaxis que se describe a continuación. ## Sintaxis Para declarar una nueva constante en el código, use la construcción `CONST..END`:\ \ `CONST`\ ` = `\ `END`\ \ Un nombre constante es cualquier identificador permitido (una combinación de letras, números y `_`). Hay nombres reservados por el compilador que no se pueden usar, como `Continue`, `Break`, `And`, etc. (mire `compiler.ini`). Un valor constante puede ser un número (también un [identificador de modelo](data-types.md#model-names) o una [etiqueta](data-types.md#labels)); a [string literal](data-types.md#string-literals); una [variable](variables.md) (también una [propiedad de clase] (classes.md#properties)); otra constante. ```sb3 var $PLAYER_CHAR: Player end const MoneyRequired = 30 PlayerMoney = $PLAYER_CHAR.Money end if PlayerMoney > MoneyRequired then PlayerMoney += -1 end ``` Durante la compilación, la constante `MoneyRequired` se sustituye por el número `30` y `PlayerMoney` con `$PLAYER_CHAR.Money`. Cuando el [servicio de idioma](../editor/language-service.md) está habilitado, se muestra una lista de constantes después de presionar`Ctrl+Space`. ## Limitaciones Puede utilizar constantes en cualquier lugar excepto en el caso: ```sb3 const VarName = $Var IndexName = 25 end VarName[IndexName] = 0 ``` Para compilar dicha expresión, debe escribir el opcode correspondiente, por ejemplo: ```sb3 0004: VarName[IndexName] = 0 ``` También hay algunas limitaciones con el uso de una expresión como valor constante.
