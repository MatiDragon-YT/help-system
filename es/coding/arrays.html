Un **arreglo** representa una colección indexada de elementos del mismo tipo (llamada en el tipo de base). Puede trabajar con cualquier elemento directamente a través de su índice. Enumerada desde el 0. Las mismas solo estan soportadas en San Andreas, LCS y VCS. ## Sintaxis General San Andreas: ```sb3 <,nombre>(<,indice>,<,tamaño><,tipo>) ``` ```sb3 $index = 0 $array($index,10i) = 1 ``` ``: una [variable](variables.md) local o global\ ``: cualquier variable que contenga el índice del elemento para leer o escribir\ ``: cualquier valor entre 1 y 255\ ``: uno de los caracteres `i` `f` `s` `v`: |Letra|Tipo de elemento|Tamaño del elemento (bytes)| |-| |i|número entero|4| |f|número decimal|4| |s|cadena corta|8| |v|cadena larga|16| :::warning Sanny es muy estricto con la sintaxis, así que no intentes poner espacios entre los corchetes o por la coma, como lo harías en cualquier lenguaje de programación. ::: Liberty City Stories, Vice City Stories:\ `(,)` :::info En LCS, VCS, los elementos de un arreglo tienen solo 4 bytes de longitud. Por lo tanto, no es necesaria una declaración de tipo. ::: ```sb3 $index = 0 $array($index,10) = 1 ``` ## Declaración de Arreglo Las matrices pueden declararse utilizando la palabra clave `var` (similar a [variables](variables.md)): ```sb3 var : array of ``` ```sb3 var $FloatArray: array 10 of Float ``` ### Accediendo a los elementos del arreglo después de la declaración Después de declarar un arreglo, puede acceder a sus elementos utilizando corchetes: ```sb3 var $FloatArray: array 10 of Float $index = 0 // aumentar el valor del primer elemento en 100 $FloatArray[$index] += 100.0 // tambien $FloatArray[0] += 100.0 ``` Se puede acceder a los elementos de un arreglo que contiene [cadenas](data-types.md#cadenas-literarias) usando [variables de cadenas](data-types.md#variables-de-cadenas): ```sb3 // inicialización de los tres primeros elementos de s$strings s$strings[0] = 'str1' s$strings[1] = 'str2' s$strings[2] = 'str3' ``` Un arreglo se puede declarar como una colección de [instancias de clase](classes.md#instancias-de-clase), sus métodos están disponibles para cada elemento:: ```sb3 var $players: array 2 of Player $players[0].Build ``` :::info Por defecto, el desensamblador imprime los elementos del array con los números como índices en el `GTA SA` [edit mode](../edit-modes/). La misma función está disponible en LCS y VCS pero está desactivada por defecto. Puede activar y desactivar esta función utilizando la opción [debug option](../editor/console.md#constant\_indexes) `CONSTANT_INDEXES`. :::
