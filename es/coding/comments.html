Un **comentario** comienzan con una barra doble `//`. Luego todo al final de la línea es ignorado por el compilador. ```sb3 // comentarios aqui 0001: wait 0 ms ``` Para comentar varias líneas o una parte de la línea, use comentarios de bloque `{}`: ```sb3 0001: wait {comentarios aqui} 0 ms ``` Desde la versión 3.06, Sanny Builder también admite comentarios similares a C++ `/* */`. ```sb3 0000: nop /* comentarios aqui */ 0001: wait 0 ms ``` Para comentar o descomentar varias líneas seleccionadas del código, presione `Ctrl+Q` (esta tecla de acceso rápido se puede configurar en las [opciones](../editor/options/hotkeys.md)).
