:::right [h2][CLEO 3] :::light gray p-2 Qué es CLEO 3? ------- ::: ::: CLEO 3 es el siguiente paso en el desarrollo del proyecto CLEO que amplia las posibilidades de codificacion en San Andreas. Despues de instalar CLEO 3, obtienes todo lo que CLEO 2 puede, mas algunas caracteristicas nuevas. 1. [Nuevos opcodes](opcodes.md).
CLEO 3 tiene 40 opcodes útiles almacenados, como administración de archivos, manejo de memoria de juegos, capacidad para llamar a funciones exe y mucho más. La tercera versión [agrega tales opcodes](opcodes.md) como una llamada de una función SCM, creando un hilo o iniciando una misión desde un archivo externo y más. 2. [Plugins](#plugins "(complementos)").
Lea más acerca de ellos a continuación. 3. [Scripts personalizados](scripts.md "(guiones)").
CLEO 3 permite agregar los scripts personalizados al juego escritos en Sanny Builder sin necesidad de iniciar un nuevo juego. Todo lo que necesita es escribir un script, colocarlo en la carpeta CLEO y ejecutar el juego (nuevo o guardado). :::right light gray p-2 ¿Qué versiones de juegos son compatibles? ------- ::: CLEO 3 es compatible con las versiones 1.0 y 1.01 de SA. Se probó con las versiones US 1.0 y EUR 1.01. Si tiene otra versión, hágame saber si CLEO 3 funciona o no para usted. :::right light gray p-2 ¿Cómo instalar CLEO 3? ------- ::: Para instalar CLEO 3, ejecute Sanny Builder 3. Verá la palabra CLEO en la barra de estado resaltada en rojo. Haga clic en él, lea el acuerdo de licencia y presione el botón 'Instalar'. Físicamente, CLEO 3 usa 3 archivos en el trabajo: vorbisFile.dll, vorbishooked.dll, cleo.asi. Siempre puede encontrar las últimas versiones de estos archivos en el sitio web [cleo.sannybuilder.com](http://cleo.sannybuilder.com/). Si usó CLEO 2 o CLEO 1, elimínelo manualmente para evitar conflictos con la nueva versión. Si usó el cleo.asi de la versión 2, reemplácelo con el nuevo. Si usó cleo.dll ejecutándose desde main.scm, debe eliminar cleo.dll y eliminar su código de carga de main.scm. Las librerias CLEO 2 y CLEO 3 son totalmente compatibles. Todas los mods/plugins escritos para CLEO 2 deberían funcionar con la nueva versión. :::right light gray p-2 ¿Debo cambiar mi SASCM.INI? ------- ::: Sanny Builder 3 tiene los archivos SASCM.INI y OPCODES.TXT ya actualizados. Las actualizaciones futuras de la biblioteca CLEO pueden requerir algunos cambios de estos archivos. Recibirá todas las instrucciones junto con las actualizaciones en [cleo.sannybuilder.com](http://cleo.sannybuilder.com/). :::right light gray p-2 ¿Qué pasa con los plugins? ------- [plugins][] ::: Los plugins son los archivos DLL con extensión .CLEO. Al inicio, la biblioteca CLEO busca en la carpeta GTASA\CLEO los archivos con esa extensión e intenta cargarlos. Si tiene éxito, el plugin comienza a funcionar y completa las funciones para las que fue creado. La instalación del plugin es muy simple: simplemente copie el archivo del plugin en la carpeta GTASA\CLEO. PARA cancelar el uso del plugin, elimine el archivo adecuado. Puede encontrar la lista de plugins disponibles en [cleo.sannybuilder.com](http://cleo.sannybuilder.com/). :::right light gray p-2 ¿CLEO 3 es redistribuible? ------- ::: Si escribió un script utilizando la biblioteca CLEO y (o) sus complementos, puede distribuir los archivos necesarios junto con ese script. También puede dirigir a los usuarios del script en [cleo.sannybuilder.com](http://cleo.sannybuilder.com/) para informarles cómo usar CLEO. :::right light gray p-2 ¿Créditos? ------- ::: El autor de la biblioteca CLEO es [Seemann](http://www.gtaforums.com/index.php?showuser=49411).
