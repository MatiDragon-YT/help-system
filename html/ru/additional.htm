Дополнительные команды

INC

Увеличивает значение переменной, переданной как первый параметр, на значение второго параметра.

Inc($IntVariable, $Value) => $IntVariable += $Value

Второй параметр может не использоваться, тогда вместо него будет использовано 1.

Inc(1@) => 1@ += 1

Начиная с версии 3.06, можно использовать C++-подобный оператор ++, который увеличивает значение переменной на 1. 

$var++ => $var += 1

DEC

Уменьшает значение переменной, переданной как первый параметр, на значение второго параметра.

Dec($IntVariable, $Value) => $IntVariable -= $Value

Второй параметр может не использоваться, тогда вместо него будет использовано 1.

Dec(1@) => 1@ -= 1

Начиная с версии 3.06, можно использовать C++-подобный оператор --, который уменьшает значение переменной на 1. 

$var-- => $var -= 1

MUL

Умножает значение переменной, переданной как первый параметр, на значение второго параметра.

Mul($IntVariable, $Value) => $IntVariable = $IntVariable * $Value

Второй параметр может не использоваться, тогда вместо него будет использовано 2.

Mul(1@) => 1@ = 1@ * 2

DIV

Делит значение переменной, переданной как первый параметр, на значение второго параметра.

Div($IntVariable, $Value) => $IntVariable = $IntVariable / $Value

Второй параметр может не использоваться, тогда вместо него будет использовано 2.

Div(1@) => 1@ = 1@ / 2

ALLOC

Указывает компилятору под каким индексом скомпилировать глобальную переменную с текстовым именам (например, таким как $text). Для переменных с числовыми именами не имеет эффекта.

Alloc($MyVar, 40) => переменная $MyVar будет скомпилирована как $40

Смотри также 'help\examples\alloc.txt'

SQR

Вычисляет квадрат значения переменной, переданной в качестве параметра, и записывает результат в эту переменную.

sqr($var) => $var *= $var

Тип переменной должен быть объявлен. Смотри раздел Переменные для дополнительной информации.

RANDOM

Возвращает случайное число в интервале, определенном параметрами.

$rnd = random(1, $high)

Данную функцию можно использовать как для типа Integer, так и для типа Float. Выбор опкода будет зависеть от типа объявленной переменной, в которую записывается случайное значение ($rnd)
