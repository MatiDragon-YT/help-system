**Ключевое слово** - это замена номеру опкода. Например, ключевое слово `WAIT` заменяет опкод `0001`. Таким образом, ```sb3 wait 0 ``` и ```sb3 0001: 0 ``` представляют собой идентичный код. Список ключевых слов содержится в файле `keywords.txt` \(свой для каждого [режима редактирования](../edit-modes/README.md)\). В редакторе его можно вызвать нажатием `Ctrl+пробел`.
