:::right [h2][CLEO 3] :::light gray p-2 Что такое CLEO 3? ------- ::: ::: CLEO 3 – это следующий шаг в развитии проекта CLEO, расширяющего границы возможностей скриптинга San Andreas. После установки библиотеки CLEO вы получаете все то, что может библиотека CLEO 2, а также некоторые нововведения третьей версии. 1. [Новые опкоды](opcodes.md).
CLEO 3 имеет в наличии 40 полезных опкодов, среди которых есть опкоды чтения файлов, изменения игровой памяти, вызова процедур и функций и многое другое. Третья версия [добавляет такие опкоды](opcodes.md) как вызов функции в SCM, запуск потока или миссии из внешнего файла и другие. 2. [Плагины](#plugins).
Более подробнее о плагинах смотри ниже. 3. [Добавление скриптов](scripts.md).
CLEO 3 позволяет добавить в игру новые скрипты, написанные в Sanny Builder’е без необходимости начала новой игры. Все, что требуется, это написать скрипт, поместить его в папку CLEO и запустить игру (новую или сейв). :::right light gray p-2 Какие версии игры поддерживаются? ------- ::: CLEO 3 supports SA versions 1.0 and 1.01. It was tested with the US 1.0 and EUR 1.01 versions. If you have another version, please, let me know if CLEO 3 works or does not for you. :::right light gray p-2 How to install CLEO 3? ------- ::: To install CLEO 3 run Sanny Builder 3. You will see word CLEO in the status bar outlined in red. Click it, read the license agreement and press the button ‘Install’. Physically, CLEO 3 uses 3 files in work: vorbisFile.dll, vorbishooked.dll, cleo.asi. You always can find latest versions of these files on the website [cleo.sannybuilder.com](http://cleo.sannybuilder.com/). If you used CLEO 2 or CLEO 1, delete it manually to avoid conflicts with new version. If you used the cleo.asi of version 2, replace it with the new one. If you used the cleo.dll running from the main.scm, you must delete the cleo.dll and remove its loading code from the main.scm. The libraries CLEO 2 and CLEO 3 are fully compatible. All mods/plugins written for CLEO 2 should work with the new version. :::right light gray p-2 Should I change my SASCM.INI? ------- ::: Sanny Builder 3 has the files SASCM.INI and OPCODES.TXT already up to date. Future updates of the CLEO library may require some changes of these files. You will get all instructions along with the updates on [cleo.sannybuilder.com](http://cleo.sannybuilder.com/). :::right light gray p-2 What about plugins? ------- [plugins][] ::: The plugins are the DLL files having an extension .CLEO. At startup, the CLEO library scans the folder GTASA\CLEO for the files with that extension and tries to load them. If successful, the plugin begins to work and complete the functions it was made for. The plugin installation is very simple: just copy the plugin file to the folder GTASA\CLEO. TO cancel the plugin usage, delete proper file. You may find the list of available plugins on [cleo.sannybuilder.com](http://cleo.sannybuilder.com/). :::right light gray p-2 Is CLEO 3 redistributable? ------- ::: If you wrote a script using the CLEO library and (or) its plugins, you may distribute needed files along with that script. You also may direct the script users on [cleo.sannybuilder.com](http://cleo.sannybuilder.com/) to inform them how to use CLEO. :::right light gray p-2 Credits? ------- ::: The author of the CLEO library is [Seemann](http://www.gtaforums.com/index.php?showuser=49411).
