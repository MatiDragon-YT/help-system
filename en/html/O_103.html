Editor

Here you can change the IDE settings.

Colors

The following options allow to set up different colors for the editor.

Active line - color of the line where the cursor is

Gutter - color of the padding area on the left to the code area of the editor

Editor - color of the code area background

Common options

The looking depth value is used while searching constants, labels and variables through the code when you press the Ctrl + [space] combo. This value tells the editor how much lines to scan up and down from the active line. If you set a number of 0, the editor will scan the whole source file.

If the 'Auto list' option is checked, the list of available labels, models and variables appears automatically when you type symbols @, $, #.

Show main panel � toggle the main toolbar

Show opcode info � make the editor display important information in the status bar (number of expecting parameters for an opcode, model ID, variable type, etc)

Line numbers � show line numbers in the left padding area

Show gutter border � display the border line on the right side of the left padding area. Uncheck this option to increase the available space visually.

Load last closed file at startup � open the last closed file at startup

Load all closed files � open all closed files at startup. This option is only availabel when the Load last closed file at startup is checked.

Confirm when exit � confirm exit to prevent occasional program termination.

Fonts

Here you can choose the font and tune it. Sanny Builder supports only fixed-pitch fonts. The line AaBbCcDd reflects the chosen settings.
